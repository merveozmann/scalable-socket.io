
# Scalable Socket.io

#### Uygulama Çalıştırma ve Test Etme

Web uygulamasını çalıştır

```sh
cd client
npm start
```
Server uygulamasını çalıştır

```sh
cd server
npm start
```
Server birim testleri çalıştır

```sh
cd server
npm test
```
